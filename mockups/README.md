# Mockups — 次期ブランディング/イントロ刷新の候補

現行の `whatif-ep.xyz` 本体（M5/M6 カットオーバー）とは**独立**した、スタンドアロンの静的モックアップ集。
Next.js アプリには一切組み込まれておらず、本番ルーティングにも影響しない。M5/M6 完了後の次期リニューアル候補として、いつでも見返せる状態でここに保管する。

各フォルダは自己完結（外部パス参照・imagine リポジトリ参照なし）。ブラウザで直接開くか、フォルダ直下で静的サーバーを立てて確認する。

```bash
cd mockups/intro-scroll && python3 -m http.server 8899 --bind 127.0.0.1
# → http://127.0.0.1:8899/short.html
# → http://127.0.0.1:8899/long.html

cd mockups/character-archive && python3 -m http.server 8898 --bind 127.0.0.1
# → http://127.0.0.1:8898/
```

## 1. `intro-scroll/` — 動画スクラブ型イントロ

スマホの縦スクロール量を動画のフレーム進行に直結させる、映画的なスクロールストーリーテリング。`position: sticky` + Canvas 連番フレーム描画（WebP）で、`video.currentTime` のスクラブ制御を避けて滑らかさを担保している。

- **`short.html`** — 6秒の縦動画（社内素材）を180フレームでスクラブ + 環境ループ映像 + IMAGINE 導線のコピー。3幕構成のティザー版。
- **`long.html`**（**WHAT THE LAND**） — 92秒のショートフィルムを12fps・1107フレームに間引き、低解像度リール全量＋高解像度ストリーミング（再生ヘッド前後の窓を先読み）の2層構成でロード。**ギャラリーへの入口（ヘッダー固定ピル + ローダーのSkip + フィナーレCTA）を常時表示**し、フィルムはブロッカーにならない設計。フィナーレの CTA は `../character-archive/index.html` に接続（下記2と繋げて見る想定）。

どちらも初回ロードは「必要な分だけ先に読み、残りは裏で流し込む」方式。長尺版は合計 3,000 枚超のフレーム画像を扱うため、確認は Wi-Fi 環境を推奨。

## 2. `character-archive/` — ゲームUI風キャラクターアーカイブ

**採用方向がほぼ固まっている最有力案。** ギャラリー本体より手前に置く、ブランディング専用のキャラクター紹介ページ（RPGのパーティ編成/ステータス画面 UX）。

- 横スワイプ/矢印キー/トラックパッド横スクロールでキャラクターを次々に入れ替え可能（5秒ごとの自動送りあり）。切り替えのたびに**キャラ固有のアクセントカラーへ画面全体（背景動画・斜めパネル・EQ・カウンター）が染まる**。
- キャラクターは IMAGINE 側で切り出し済みの `character_cutout` アセット（`imagine/video/imagine-promo/public/char/` から8体を選定）をこのフォルダにコピー済み。ロア・ステータス（SYNC/NOISE/VOLT）はこのモック用に新規で書き下ろし。
- 右上に BGM トグル（EQバー演出、デフォルト OFF、押した瞬間だけ mp3 を読み込む）。
- 右上のショップアイコンから**壁紙購入モードのオーバーレイ**（ゲームのアイテム選択/装備画面 UI）を起動。カード型カルーセル + 長押し購入（Hold to Buy）。カバーアートは3点の実素材 + ダミー2点 + ロック済み2枠。
- 各キャラの「View in Gallery →」・購入完了後の遷移先は現状 `https://whatif-ep.xyz`（将来的にはキャラ/タグ別のギャラリー検索結果に接続する想定）。
